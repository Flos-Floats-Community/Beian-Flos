# 浮云社 ICP备案查询系统

## 项目概述
这是一个基于PHP和SQLite的ICP备案查询系统，具有用户注册/登录、备案信息管理、举报功能和管理员后台等功能。

## 功能特点
- 用户注册/登录系统
- 备案信息查询（网站名称、域名、主体等）
- 备案申请与审核流程
- 主体验证等级管理
- 举报功能
- 管理员后台

## 数据库表结构
- `users` 表：存储用户信息
- `beian_info` 表：存储备案信息

## 安装说明

1. 确保服务器环境支持：
   - PHP 7.0+
   - SQLite 支持
   - Apache 或 Nginx

2. 将项目文件上传到Web服务器

3. 初始化数据库：
   ```bash
   php database/init_db.php
   ```

4. 设置目录权限：
   - 确保 `database/` 目录可写

## 使用说明

### 前台功能
- 访问 `public/index.php` 进行备案查询
- 用户可通过 `public/register.php` 注册
- 用户可通过 `public/login.php` 登录
- 可对备案信息进行举报

### 后台管理
- 管理员访问路径：`/admin/flos`
- 默认管理员账号：`flos`
- 默认管理员密码：`Flos123456`
- 管理员可审核备案申请、更新验证等级

### 默认管理员
- 用户名：`Admin`
- 密码：`Flos123456`

## 伪静态配置
项目包含 `.htaccess` 文件以支持URL重写。

## 字段说明
| 字段 | 说明 |
|------|------|
| 网站名称 | 网站的名称 |
| 网站域名 | 网站的域名 |
| 网站首页 | 网站的首页地址 |
| 网站信息 | 网站的相关信息 |
| 备案号 | 浮备案号 |
| 国家/其他备案社区备案号 | 社区备案号（可选） |
| 备案社区归属 | 备案社区归属（可选） |
| 所有者 | 网站所有者 |
| 主体 | 网站主体 |
| 更新时间 | 最后更新时间 |
| 状态 | 审核状态 |
| 是否可以验证主体 | 主体危险程度 |
| 举报 | 举报功能 |
